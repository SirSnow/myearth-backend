不忘初心，方得始终


迈尔斯国际在努力探索全球顶尖名校申请的同时，也秉承着一切从用户需求出发的初心，同时致力于将更多优质的国际顶尖教育体系产品带给寻常百姓家。我们自信以迈尔斯国际资深的申请经验和名师团队，将会给全球顶尖名校输送出更多优质的生源，这也是迈尔斯实现高端留学普及化源源不断的动力。

首先我们要最诚挚的祝贺，祝贺勇敢的你，祝贺无畏的你，祝贺有勇气追梦的你，祝贺你选择了留学，选择了自己人生崭新的可能。 然后还要祝贺你，祝贺幸运的你，祝贺明智的你，在众多的选择中，选择了迈尔斯团队，为你的留学之路保驾护航，迈尔斯，致力于高端留学平民化，把更多的机会带给信赖我们的你。 即将到来的异国的生活或许让你兴奋，让你期待满满，也让充满好奇，或许还有一丝忐忑，不安，不要着急，迈尔斯团队将全程为你服务。

一，初步评估

你目之所及的对手，只是茫茫海域的冰山一角，迈尔斯用专业的初步评估系统，助你在航行中，完美躲避暗礁，打败竞争对手，安全顺利着陆。

二,深度评估

“认识你自己”是人类永恒的课题。通过迈尔斯专业团队的自我认知系统，你将会前所未有的认识自己，理解自己，欣赏自己。30至50小时的深度了解，让你在文书和推荐信中展现一个与众不同，耀眼夺目的自己。

三，录取研讨会

“知己知彼，百战不殆”，迈尔斯用庞大的数据库，专业的业务能力，细致的调查沟通，对你进行为期一周，每天至少5小时的深度培训，让你轻松赢在起跑线。

四，通用申请和附加文件

选择了远方，便只顾风雨兼程。确定学校名单后，我们将根据对你的深入了解，使用头脑风暴法，写出你独一无二的故事。在这个过程中，你也会通过大量的文稿改写（平均每人100篇），遇见更优秀的自己。

五，面试

如何让考官喜欢你？第一步：由衷的欣赏自己。第二步：充分的展现自己。 迈尔斯将通过口语培训，技巧指导，实战模拟等多种方式，让你提升自信，培养能力，轻松展现自己的最佳状态。

六，录取，签证及行前指导

此时，你可以尽享成功前的期待和喜悦，而迈尔斯团队，仍将以坚持不懈的精神为你值好最后一班，确定最终学校后，我们将会全程指导你，准备各项签证及入学材料，行前准备，及选课指导。 我们相信，通过以上一系列的流程，迈尔斯将让你全程无忧的轻松开启留学新生活。 迈尔斯，始终秉持初心，为你一路守护。
